### CAFE SIMULATOR APP

#### 1. Start react app

```
yarn install
yarn start
```

#### 2. Start strapi backend in a new terminal

```
cd ./thorsystems-test-backend
yarn install
yarn start
# Database dump located inside .tmp folder
# API keys are included as well...
```

#### 3. Open browser

```
http://localhost:1337 - strapi admin
http://localhost:3001 - react app
```
